# 🎰 Slot Game Prototype (PixiJS + React)

This project is a small slot‑machine prototype built with **PixiJS** for rendering and animation, and **React** for the HUD (balance, win display, and spin button).

The goal is to keep the architecture simple and readable while still behaving like a real slot game:
- reels spin independently
- reels slow down and stop smoothly
- wins are calculated only after all reels stop
- UI reacts to game state changes
